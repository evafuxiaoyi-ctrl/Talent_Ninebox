# 人才盘点 Excel 整合工具使用说明

## 1. 访问方式

工具地址：

```text
https://talent-ninebox-web.vercel.app
```

访问密码：

```text
TalentNinebox@2026
```

## 2. 工具用途

本工具用于支持人才盘点校准前后的两阶段工作流：

1. 初版整合：HRBP 回收各部门填写的模板后，上传 zip，系统自动合并数据，并按「初始九宫格落位」生成初版人才九宫格。
2. 终版生成：校准会后，HRBP 在整合结果中填写「校准后九宫格落位」，再上传整合后的 Excel，系统按终版落位重新生成九宫格。

工具不做人选判断，只做 Excel 自动合并、九宫格生成和格式保留。

## 3. 两种处理模式

### 3.1 初版整合

适用场景：

```text
各部门刚回收模板，终版落位尚未填写。
```

上传文件：

```text
.zip
```

zip 内要求：

- 最多 30 个 `.xlsx`
- 各文件使用同一份人才盘点模板
- 可以包含文件夹，系统会递归识别
- 不支持 `.xls`、加密 Excel、损坏 Excel

九宫格依据：

```text
初始九宫格落位
```

如果模板中的「初始九宫格落位」是公式列，系统会根据绩效和能力评分推导初版落位，并写入结果文件。

### 3.2 终版生成

适用场景：

```text
校准会结束后，HRBP 已在整合结果中填写校准后的终版落位。
```

上传文件：

```text
.xlsx
```

九宫格依据：

```text
校准后九宫格落位
```

## 4. 操作步骤

1. 打开工具地址。
2. 输入访问密码。
3. 选择处理模式：
   - 初版整合
   - 终版生成
4. 上传对应文件。
5. 点击「开始处理」。
6. 处理完成后点击「下载 Excel」。

处理完成后请及时下载结果文件。

## 5. 输出文件

初版整合输出：

```text
人才盘点整合结果_初版_YYYYMMDD_HHMMSS.xlsx
```

终版生成输出：

```text
人才盘点整合结果_终版_YYYYMMDD_HHMMSS.xlsx
```

输出文件包含：

| Sheet | 说明 |
|---|---|
| 使用说明 | 结果文件内说明 |
| 参数 | 原模板参数表，用于保留公式和下拉框引用 |
| 整合总表 | 合并后的完整人员数据 |
| 人才九宫格 | 可编辑的人才九宫格 |
| 处理摘要 | 文件数量、人员数量、落位数量等汇总 |
| 异常报告 | 文件、模板、数据异常明细 |
| 文件来源映射 | 每条数据对应的原始文件和行号 |

## 6. 整合总表保留规则

整合总表会尽量保留原模板结构：

- 模板顶部标题行
- 表头行
- 参数 Sheet
- 公式
- 下拉框
- 列宽和基础样式

合并后，「序号」列会重新生成：

```text
1, 2, 3, ...
```

## 7. 九宫格规则

九宫格横轴：

```text
绩效：低 → 中 → 高
```

九宫格纵轴：

```text
能力/潜力：高 → 中 → 低
```

人员展示格式：

```text
姓名-一级部门-二级部门
```

九宫格是 Excel 单元格文本，不是图片，可直接编辑、截图或复制到 PPT。

## 8. 颜色标记规则

以下规则在「整合总表」和「人才九宫格」中都会生效。

价值观综合得分：

- `>= 4`：标记为绿色
- `<= 2`：标记为红色

离职风险：

- 离职风险为「高」：标记为红色

在人才九宫格中：

- 价值观高分人员：姓名所在单元格标绿
- 价值观低分人员：姓名所在单元格标红
- 离职风险高人员：姓名字体标红

## 9. 常见问题

### 9.1 初版整合时，终版落位为空可以吗？

可以。初版整合使用「初始九宫格落位」，不要求填写「校准后九宫格落位」。

### 9.2 终版生成时需要上传 zip 吗？

不需要。终版生成上传校准会后填写完成的整合结果 `.xlsx`。

### 9.3 为什么公式结果需要打开 Excel 后刷新？

工具会保留公式，但不会像 Excel 一样计算所有公式结果。打开结果文件后，Excel 会自动刷新公式。

### 9.4 为什么会出现未落位人员？

常见原因：

- 对应阶段的九宫格落位为空
- 落位值不是标准格式
- 人员行缺少姓名或工号

异常会记录在「异常报告」中。

### 9.5 当前版本是否保存历史记录？

不保存。处理结果只用于即时下载。

## 10. 使用建议

- 初版整合前，确认各部门使用同一模板。
- 初版整合上传 zip，不要上传单个 xlsx。
- 校准会后，在整合结果里填写「校准后九宫格落位」。
- 终版生成上传填写后的整合结果 xlsx。
- 每次处理完成后及时下载结果文件。
